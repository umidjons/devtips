# Basics

## Windows

`^b-c` - create new window
`^b-,` - rename the window
`^b-w` - list available windows
`^b-p` - previous window
`^b-n` - next window

## Panes

`^b-%` - split vertically
`^b-"` - split horizontally

## Sessions

`tmux new -s <session name>` - create a new named session
`^b-d` - detach from the session
`^b-$` - rename a session
